Tecnologías utilizadas

  • Node.js
  • Express.js
  • dotenv
  • nodemon
  • HTML5 y CSS3

Arquitectura implementada

El proyecto está organizado utilizando una estructura modular que separa rutas, controladores, middlewares, servicios y archivos públicos. Esto permite una mejor mantención, escalabilidad y claridad del código.

Funcionalidad del sistema

El sistema responde a rutas web, entrega información en formato HTML y JSON, y registra cada acceso en un archivo de texto para evidenciar el uso del servidor.
