Descripción del proyecto

Esta aplicación fue desarrollada como parte del Proyecto ABP del Módulo 6, utilizando Node.js, Express, rutas, controladores, middlewares y archivos estáticos.

Rutas disponibles

  • / Página principal
  • /about Información del proyecto
  • /contact Datos del desarrollador
  • /status Estado del servidor en JSON
  • /api/visits Registros del archivo log.txt

Características implementadas

  • Servidor Express funcionando
  • Arquitectura modular
  • Rutas separadas
  • Controladores
  • Middleware de registro de visitas
  • Servicio para lectura y escritura de logs
  • Archivos estáticos HTML y CSS

Objetivo

Demostrar la creación de una aplicación web básica con backend en Node.js y Express, incluyendo organización modular del código, manejo de rutas y registro de accesos.

Desarrollado para el Proyecto ABP - Módulo 6
